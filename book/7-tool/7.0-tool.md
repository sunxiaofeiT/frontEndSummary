# 工具

## Git

### 常用命令

|常用命令       |作用                   |
|--------------|-----------------------|
|git fetch      |拉取                    |
|git pull       |拉取合并                |
|git push       |推送                    | 
|git status     |当前状态                |
|git add .      |添加修改文件            |
|git commit -m  |暂存                    |
|git tag        |打tag                  |
|git branch     |新建分支                |
|git checkout   |切换分支、回滚           |
|git remote     |操作远程库              |
|git config     |操作git配置文件         |
|git merge      |合并                    |

### git fetch 与 git pull 的区别

一般认为， git pull 是 git fetch 和 git merge 的合并。
